How We Handle Your Data

Your privacy is my top priority. This application is designed to be a "Zero-Retention" system. Here is exactly what happens when you use this tool:

1. Resume Processing & Evaluation

When you upload a PDF, it is processed and analyzed by our AI. The transient copy of the PDF stored on our server is deleted permanently immediately after processing.

2. API Keys

The Groq API key you provide is handled entirely in-memory. It is sent as an authentication header to the Groq API and is never logged, cached, or saved to a file. Once your browser session ends, the key is gone.

3. GitHub Data

The application only fetches public information from your GitHub profile if a URL is found in your resume. This data is used solely to enhance your technical score.

4. Third-Party Services

We use the Groq API to process and score your resume. By using this tool, you acknowledge that your resume text (not the PDF file itself) is sent to Groq for analysis under their privacy terms.
